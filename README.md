## 理论学习教材

如果仅仅是为了学习数据结构与算法，其实没必要太过注意学习邓俊辉的《数据结构和算法》，C++里包含了一些特性，非cs专业的学生很难用到，但是对于计算机科班的学生是十分有必要的。

- 严蔚敏《数据结构与算法c语言版》
- 邓俊辉《数据结构与算法c++版》
- 王道《数据结构》
